# About Reaqtor 

Reaqtor is a framework for reliable, stateful, distributed, and scalable event processing based on [Rx](https://github.com/dotnet/reactive).

## .NET Foundation

Reaqtor is part of the [.NET Foundation](https://www.dotnetfoundation.org/).
